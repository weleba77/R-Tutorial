# R-Tutorial

A comprehensive guide and reference repository for learning R programming basics, data manipulation, file handling, and package usage.

## 📂 Repository Structure

*   `codtion.R` - Introduction to conditional statements (`if/else`), loops (`for`), and custom functions.
*   `intro to R.R` - Basic R syntax, variables, data types, and fundamental concepts.
*   `ReadAndWriteInToCSV.R` - Scripts demonstrating how to import and export data using CSV files.
*   `pakage.R` - Core package installation and loading demonstrations.
*   `pakage2.R` - Data visualization and manipulation examples using `ggplot2` and `dplyr`.
*   `data.R` - General data structures, vectors, data frames, and matrices in R.
*   `tested.csv` / `titanic_old_survived.csv` - Sample datasets used for practicing data analysis workflows.

## 🚀 Getting Started

### Installation
Clone this repository to your local machine:
```bash
git clone https://github.com
cd R-Tutorial
```

## 🛠️ Required Packages
Some scripts (like `pakage2.R`) require external libraries. You can install them by running the following commands in your R console:

```R
install.packages("dplyr")   # For data manipulation
install.packages("ggplot2") # For data visualization
```
